\section{Theoretical Foundations}

This section introduces theoretical concepts needed for understanding
\textsc{Natural}'s design and implementation sections. It covers relational
database theory, machine learning fundamentals, large language models, natural
language processing, embeddings and graph theory.

These foundations support the system's core functions: example selection
($\sigma$), schema subsetting ($\phi$), query projection ($\pi$),
self-refinement ($\rho$), and voting ($\nu$).
